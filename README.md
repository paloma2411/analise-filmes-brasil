# 🎬 Análise do Cinema Brasileiro e Estrangeiro (2009-2019)

Projeto final do curso de Técnicas de Programação.

## 📌 Sobre o projeto
Análise de **7.045 filmes** exibidos no Brasil entre 2009 e 2019, explorando padrões de público, renda, distribuição e gênero no mercado cinematográfico nacional.

## 📂 Dataset
- **Fonte:** Dados públicos do setor cinematográfico brasileiro (ANCINE)
- **Período:** 2009 a 2019
- **Colunas:** ano, título, gênero, países produtores, nacionalidade, distribuidora, origem da distribuidora, público e renda.

## 🛠️ Tecnologias
- Python 3
- Pandas
- Matplotlib (via Pandas `.plot()`)
- Jupyter Notebook (VS Code)

## ❓ Perguntas respondidas
1. Como evoluiu o público e a renda do cinema entre 2009 e 2019?
2. Filmes brasileiros competem com os estrangeiros em público e bilheteria?
3. Qual gênero cinematográfico atrai mais público em média?
4. Quais são as 10 distribuidoras mais poderosas da década?
5. Filmes brasileiros cobram mais ou menos que os estrangeiros?

## 🔍 Principais descobertas
- O público cresceu **56,6%** e a renda **187,7%** na década.
- Filmes estrangeiros dominam o mercado em público e renda.
- **Animação** tem o maior público médio por filme (~615 mil espectadores).
- **Disney** é a distribuidora líder em renda (R$ 4,3 bilhões).
- Filmes estrangeiros cobram em média **R$ 1,99 a mais** por ingresso.

## 🚀 Como executar
1. Clone este repositório.
2. Instale as dependências: `pip install pandas matplotlib ipykernel`
3. Abra o `analise_filmes.ipynb` no VS Code.
4. Execute todas as células com `Run All`.
